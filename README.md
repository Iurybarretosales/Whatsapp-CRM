# CRM Integrado com WhatsApp - Personalização de Pipeline de Vendas

Este repositório contém o código de um **CRM integrado ao WhatsApp**, permitindo que os usuários criem e personalizem suas próprias pipelines de vendas, além de automatizar a interação com clientes diretamente pelo WhatsApp.

## Funcionalidades

- **Integração com WhatsApp Web**: Envio e recebimento de mensagens diretamente pelo WhatsApp, permitindo comunicação fluída com os clientes.
- **Criação e Personalização de Pipelines de Vendas**: Permite a criação de etapas personalizadas para o funil de vendas, com total flexibilidade para adaptar à necessidade de cada negócio.
- **Automação de Processos**: Configuração de mensagens automáticas em cada etapa do funil, economizando tempo e garantindo consistência.
- **Relatórios e Análises**: Acompanhamento de métricas como taxa de conversão, tempo médio de vendas, e performance de cada pipeline.
- **Gestão de Leads**: Organização de contatos e leads, com possibilidade de categorizar conforme o estágio do funil de vendas.

## Tecnologias Usadas

- **json**: Para criação do servidor backend.
- **javascript**: Para comunicação em tempo real com o WhatsApp Web.
- **css**: Banco de dados NoSQL para armazenamento de dados de usuários, leads e mensagens.
- **html**: Biblioteca para construção da interface de usuário (front-end).
- **API do WhatsApp Web**: Comunicação com a plataforma WhatsApp Web para interação com mensagens.
